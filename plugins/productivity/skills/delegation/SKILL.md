# Delegation & Automation

keywords: déléguer, automatiser, automation, outsource, assistant, temps, efficacité, levier

## Framework de Délégation

### Matrice Eisenhower + Délégation:
| | Urgent | Pas Urgent |
|---|--------|-----------|
| **Important** | FAIRE soi-même | PLANIFIER (deep work) |
| **Pas Important** | DÉLÉGUER (Kingston/agents) | ÉLIMINER |

### Quoi Déléguer à Kingston:
- Recherche et veille (Scout agent)
- Analyse et rapports (Analyst agent)
- Prospection initiale (outreach templates)
- Gestion de contenu (pipeline Moltbook)
- Monitoring trading (alertes automatiques)
- Admin: factures, scheduling, follow-ups
- Code requests (Executor agent)

### Quoi Garder pour Soi:
- Décisions stratégiques majeures
- Relations clés (networking high-value)
- Création de contenu (voix authentique)
- Trading: décision finale d'exécution
- Vision et direction business

### Checklist Avant de Déléguer:
1. ☐ Résultat attendu clairement défini?
2. ☐ Deadline spécifiée?
3. ☐ Contexte suffisant fourni?
4. ☐ Critères de succès définis?
5. ☐ Point de contrôle planifié?

### Automation Ladder:
1. **Manuel** → Faire soi-même, documenter le process
2. **Assisted** → Kingston aide (suggestions, drafts)
3. **Semi-auto** → Kingston fait, Nicolas valide
4. **Full auto** → Kingston fait tout (crons, agents)
5. **Intelligent** → Kingston décide + fait (Mind agent)
