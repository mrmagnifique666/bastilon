# Focus & Deep Work

keywords: focus, concentration, deep work, distraction, productif, zone, flow, attention

## Protocole Deep Work

### Conditions pour le Flow State:
1. **Objectif clair**: Savoir exactement ce qu'on veut accomplir
2. **Feedback immédiat**: Pouvoir mesurer l'avancement
3. **Challenge calibré**: Ni trop facile (ennui) ni trop dur (anxiété)
4. **Zéro interruption**: Téléphone en mode avion, notifications off, porte fermée

### Techniques Anti-Distraction:
- **Timeboxing**: Bloquer des créneaux dans le calendrier (traiter comme des meetings)
- **Environment design**: Lieu dédié au travail, pas le même que pour le loisir
- **Commitment device**: Dire à quelqu'un "dans 2h j'aurai fini X"
- **Capture bin**: Noter les pensées parasites sur papier pour plus tard
- **Music**: Lo-fi, musique instrumentale, bruit blanc (pas de paroles)

### Quand Décrocher:
- Après 90min de focus intense → pause obligatoire 15-20min
- Si productivité tombe sous 50% → changer d'activité
- Après 4h de deep work dans la journée → tâches légères
- Fatigue cognitive = pas fainéant, c'est un signal biologique

### Recovery:
- Marche 20min (pas de téléphone)
- Nap 20min (pas plus, sinon groggy)
- Exercice physique (reset complet du cerveau)
- Social léger (conversation non-travail)
