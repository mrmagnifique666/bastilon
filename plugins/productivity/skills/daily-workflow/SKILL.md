# Daily Workflow

keywords: routine, workflow, matin, soir, productivité, habitudes, énergie, focus, journée

## Routine Optimale (Entrepreneur Solo)

### Morning Block (7h-9h) — Strategic:
1. **7h00**: Météo mentale — comment je me sens? (1-10)
2. **7h15**: Briefing Kingston — marché, pipeline, Moltbook, goals
3. **7h30**: Pre-market research (si trading actif)
4. **8h00**: Deep work #1 — la tâche la plus importante, 0 distraction
5. **8h45**: Répondre messages urgents seulement

### Focus Block (9h-12h) — Execution:
- **Pomodoro 50/10**: 50min focus, 10min pause
- Max 2-3 tâches majeures par bloc
- Notifications OFF (sauf Telegram urgent)
- Si bloqué >15min → changer de tâche ou déléguer

### Afternoon Block (13h-17h) — Communication:
- **13h-14h**: Calls, meetings, networking
- **14h-16h**: Tâches créatives (content, stratégie)
- **16h-17h**: Admin, emails, follow-ups, pipeline review

### Evening Block (17h-19h) — Review:
- **17h00**: Trade journal (si trading)
- **17h30**: Review des goals — avancement du jour
- **18h00**: Planifier demain (3 priorités max)
- **18h30**: Déconnexion — pas de travail après sauf urgence

### Règles d'Or:
- **Eat the frog**: La tâche la plus dure en premier
- **2-minute rule**: Si ça prend <2min, fais-le maintenant
- **Batching**: Grouper les tâches similaires (tous les emails d'un coup)
- **Parkinson**: Donner des deadlines serrées (le travail remplit le temps alloué)
- **Single-tasking**: Une chose à la fois. Le multitasking est un mythe.
