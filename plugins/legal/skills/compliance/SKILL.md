# Compliance & Propriété Intellectuelle

keywords: compliance, conformité, GDPR, vie privée, privacy, PI, propriété intellectuelle, licence, copyright, marque

## Compliance pour Entrepreneur Tech

### GDPR/Vie Privée (si clients EU):
- **Consentement explicite** pour collecter des données personnelles
- **Droit à l'effacement** — pouvoir supprimer les données sur demande
- **Privacy policy** sur le site web (obligatoire)
- **Data processing agreement** avec tout sous-traitant
- **Notification de breach** dans les 72h si violation de données
- Pas de données personnelles dans les logs sans anonymisation

### Propriété Intellectuelle:
- **Copyright**: Automatique à la création (code, texte, design)
- **Marque**: Enregistrer le nom/logo (OPIC au Canada, USPTO aux US)
- **Brevet**: Pour les inventions techniques (rarement pertinent pour du SaaS)
- **Trade secret**: Protéger via NDA et accès restreint

### Licences Open Source:
| Licence | Permissions | Obligations |
|---------|------------|-------------|
| **MIT** | Tout | Garder la notice de copyright |
| **Apache 2.0** | Tout + brevets | Notice + changements documentés |
| **GPL** | Tout | Code dérivé DOIT être GPL aussi |
| **AGPL** | Tout | Même utilisation serveur = partager le code |
| **BSL** | Usage non-commercial | Commercial après délai |

### Checklist Compliance Annuelle:
1. ☐ Privacy policy à jour?
2. ☐ Terms of service révisés?
3. ☐ Licences open source auditées?
4. ☐ Certificats SSL renouvelés?
5. ☐ Backups et disaster recovery testés?
6. ☐ Accès et permissions révisés?
