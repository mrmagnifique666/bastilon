> Why This Exists

Every API says "easy setup in 5 minutes." It's never 5 minutes.

The official docs show you the ideal path. But the real path has detours: buttons that moved, settings buried 4 menus deep, verification emails that never arrive, and error messages that mean the opposite of what they say.

This guide documents the real path. Every step was tested by walking through the actual website, clicking the actual buttons, in the actual UI as it exists right now — not as it existed when the docs were written.

Each guide includes the date it was last verified. If a step no longer works, open an issue and we'll update it.
