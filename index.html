Dual-Agent Architecture

Kingston (The Voice) handles user interaction, planning, and memory. Emile (The Hands) executes code, deploys, and builds. Together, they form a complete system.

476+ Built-in Skills

Email, SMS, trading, browser automation, social media, voice, wake word, knowledge graph, cron jobs, episodic memory, workflows, MCP server, RAG ingestion, D&D dungeon master, and more. All ready out of the box.

7+ Agent System

Scout prospects leads. Analyst reports performance. Sentinel monitors security. Kingston Mind makes autonomous business decisions. Learner, Executor, Trading Monitor. Plus on-demand subordinate agents. All running 24/7.

Self-Improving Memory

Knowledge graph, episodic memory, behavioral rules engine, semantic memory with embeddings, RAG URL ingestion, nightly council reviews. Kingston remembers, reasons, and evolves autonomously.

Multi-Platform + Gateway

Telegram, Twitter, LinkedIn, Reddit, Discord, Facebook, Instagram, WhatsApp, SMS. WebSocket gateway, MCP server for Claude Desktop. Connect everywhere your users are.

Voice + Wake Word + Gemini Live

Edge TTS + Deepgram STT + Picovoice wake word + Gemini Live cloud streaming. Say "Computer" and Kingston listens. Phone calls via Twilio. Real-time voice conversations with tool execution.
