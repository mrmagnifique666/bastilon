## ClawHub Skill Security Review Process

1.  **Inspector run** → Generate security report
2.  **Sandbox test** → Verify behavior matches description
3.  **Manual review** → Nicolas approves based on report
4.  **Install** → Copy to `src/skills/community/<skill-name>/`
5.  **Register** → Add to skills loader with security profile
6.  **Test in production** → Verify integration with Bastilon
